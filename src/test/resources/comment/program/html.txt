						<h3 class="tit"><a href="/entry/mybatis-procedure-cursor-resulthandler" class="link">mybatis procedure cursor resulthandler</a></h3>
						<div class="article">
							<p>mybatis&nbsp;procedure&nbsp;cursor&nbsp;resulthandler</p>
<p>mybatis 프로시저 커서 resulsethandler 사용은 3.4.6 버전이 후에 추가된것 같음.&nbsp;</p>
<p>사용할려면 3.4.6버전이후 버전으로 사용.&nbsp;</p>
<p>------------------------------------------------------------------------------------------------------------------------------</p>
<p>ResultHandler파라미터는 레코드별로 다룰수 있도록 해준다. List에 추가할수도 있고 Map, Set을 만들수도 있으며 각각의 결과를 그냥 던질수도 있다. ResultHandler로 많은 것을 할 수 있고 마이바티스는 결과셋을 다루기 위해 내부적으로 사용한다.</p>
<p>Since 3.4.6, ResultHandler passed to a CALLABLE statement is used on every REFCURSOR output parameter of the stored procedure if there is any.</p>
